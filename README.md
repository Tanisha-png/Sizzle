# Sizzle 🔥 👩🏽‍🍳

**Sizzle** is a full-stack digital recipe vault designed to help home cooks organize their recipes. It allows the users to manage their own personal recipes while integrating with the external food APIs to discover and save new recipes.

---

## 👁️ The Vision
Most recipe blogs are filled with ads and long stories which can be aggravating. **Sizzle** provides a clean experience. It's built for speed helping you from "What should I cook?" to "Dinner is ready" as quickly as possible.

## 🔩 Tech Stack
This project is a full-stack **MERN** application built with a focus on type safety and scalable architecture.

- **Frontend:** React, TypeScript, JavaScript, CSS
- **Backend:** Node.js, Express 
- **Database:** MongoDB (Mongoose)
- **Tools:** Git, Postman, Vite
  
## 🔑 Key Features (MVP)

- **Centralized Dashboard:** View all your saved recipes in one clean interface.
- **Full CRUD Functionality:**
  - **Create:** Add personal recipes with ingredient lists.
  - **Read:** Detailed view for instructions and cook times.
  - **Update:** Tweak and refine your recipes over time.
  - **Delete:** Keep your collection curated and clutter-free
- **API:** Search for recipes and save them directly to your personal database
